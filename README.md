# Googleのページに常時表示されている個人情報を消すChrome拡張機能

Googleのページの上部に常時表示されている住所や氏名を削除します。 \
画面共有時などで個人情報を出したくない時に便利です。

## 注意点
- 日本版のページ(`google.co.jp`, `google.com`)でしか動作しません。
  - 他のページに対応させたい場合はご自身で修正してプルリクエストを発行してください。
- `display: none;`で要素ごと表示しないようにしているので、操作上の何かしらの問題が起こりうる可能性があります。
  - `visibility: hidden;`に書き直せばテキストが消えるだけで済みますが、無駄な空白ができる可能性があります。
